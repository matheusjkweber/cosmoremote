Capacidades

Pensado para trabajar remoto sin perder foco

Del emparejamiento local al relay remoto: cambias de contexto rápido y el historial no se pierde.

Instalación con un comando

Instala el bridge daemon con brew, winget o script curl y empieza en segundos.

cosmoremote start

Detecta Claude Code y Codex, levanta el WebSocket local y muestra el QR en la terminal.

Emparejamiento por QR

Escanea una vez y conecta por IP local con token autenticado.

Selector de CLI

Elige Claude Code o Codex por sesión antes de enviar comandos.

Selector de directorio

Guarda proyectos con nombres claros y cambia de contexto al instante.

Streaming en tiempo real

Salida token a token con auto-scroll y sin polling.

Copiar salida

Copia la sesión entera o un bloque de código con un toque.

Estado de conexión

Indicadores claros para local, remoto y desconectado mientras trabajas.
